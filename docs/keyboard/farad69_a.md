Farad69 REV.A 键盘
=====================
本页面仅描述REV.A版本相关信息

部分老旧PCB上没有版本标识，属于Rev.A版本

## 描述

- 5x15的60%键盘，40%+Pad的布局
- 一个Caps灯和三个单色状态指示灯
- Type-C接口
- 蓝牙主控为nRF52832（信驰达RF-BM-ND04蓝牙模块）
- 采用热拔插方式（佳达隆轴座）
- 预留WS2812 RGB灯带焊接位
- PCB尺寸：285mm×95.3mm
