GT BLE60 rev.D 键盘
=====================
本页面仅描述rev.D版本（无灯）相关信息。

部分老旧PCB上没有版本标识，属于Rev.D版本

## 描述

- 5x14的60%键盘
- Type-C接口
- 蓝牙主控为nRF52832（信驰达RF-BM-ND04蓝牙模块）
- 采用热拔插方式,同时支持焊接轴体
- 状态指示灯3颗
- Caps指示灯1颗
- 预留WS2812 RGB灯带焊接位
- PCB尺寸：285mm×95.3mm
