Omega50 Rev.A 介绍
=====================
本页面仅描述rev.A版本（轴灯版）相关信息

部分老旧PCB上没有版本标识，属于Rev.A 版本

描述
---------

- 5x12的50%键盘
- 蓝牙主控为nRF52832（信驰达RF-BM-ND04蓝牙模块）
- 采用热拔插方式
- 状态指示灯3颗
- Caps指示灯1颗
- 充电指示灯2颗
- 8颗RGB底灯（WS2812）
- 预留WS2812 RGB灯带焊接位
