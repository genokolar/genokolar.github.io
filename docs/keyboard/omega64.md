Omega64 键盘
=====================

左移64版本（Rev.A） 与 非左移64版本（Rev.B）共用同一个固件，在配置工具--键盘设置中选择左移与非左移既可以进行区别。

Omega64 rev.A
------------

- 5x14的60%键盘，左移64版本
- Type-C接口
- 蓝牙主控为nRF52832（信驰达RF-BM-ND04蓝牙模块）
- 采用热拔插方式（佳达隆轴座）
- 状态指示灯3颗
- Caps指示灯1颗
- 8颗RGB底灯（WS2812）
- 预留WS2812 RGB灯带焊接位
- PCB尺寸：285mm×95.3mm

Omega64 rev.B
------------

- 5x14的60%键盘，非左移64版本
- Type-C接口
- 蓝牙主控为nRF52832（信驰达RF-BM-ND04蓝牙模块）
- 采用热拔插方式（佳达隆轴座）
- 状态指示灯3颗
- Caps指示灯1颗
- 8颗RGB底灯（WS2812）
- 预留WS2812 RGB灯带焊接位
- PCB尺寸：285mm×95.3mm
