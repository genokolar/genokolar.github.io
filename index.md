# 公告

1. 部分新键盘已经提供基于nRF SDK17.0.2的固件，请进入对应产品页面进行下载。

2. 基于nRF SDK17.0.2的固件不能与基于nRF SDK15.3的固件混用，如需两者间切换，请刷写蓝牙完整固件进行切换。

3. 2.4G无线功能已经提上开发日程，升级完成后，所有新老硬件均可支持。

4. 因USB无需更新，USB固件不再提供下载，如确需下载的请联系我们。
